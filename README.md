# MusicPlayerDome
音乐播放器<br>
模仿网易云音乐（个人练习 制作的音乐播放器）<br>
完成API大多数功能，可正常使用<br>
采用了NeteaseCloudMusicApi（ https://github.com/Binaryify/NeteaseCloudMusicApi ）<br>
